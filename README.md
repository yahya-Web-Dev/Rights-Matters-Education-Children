# Rights-Matters-Education-Children

A responsive website presenting student and child rights. Built in HTML and CSS. The website is written in Arabic and showcases a clean, mobile-friendly design for educational purposes.

Features:

Fully responsive layout with desktop, tablet, and mobile views.

Organized sections: Child Rights and Student Rights.

Clean and readable Arabic typography.

No JavaScript, only HTML and CSS for simplicity.

Competition Context:

Created for a school competition where participants represent their ideas through a project. The site highlights rights for students and children in a clear and engaging manner.

Project Highlights:

Completed in 4 days out of a 7-day deadline.

Original CSS and HTML authored manually.

Media queries written with AI assistance to ensure proper responsiveness.
